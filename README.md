## 🏁 Overview
**AERO** is a high-end, responsive landing page designed for the luxury electric vehicle market. It focuses on minimalist design, high-contrast visual storytelling, and smooth micro-interactions.

### 💎 Key Features
* **Immersive Hero:** Parallax-driven video/image background with "reveal" entrance animations.
* **Dynamic Fleet:** Real-time JavaScript filtering for different car categories (SUV vs Sport).
* **Scrollytelling:** Scroll-activated navigation transitions and parallax depth effects.
* **Performance Metrics:** Clean, high-impact data visualization for vehicle specifications.
* **Fully Responsive:** Fluid layouts optimized for mobile, tablet, and ultra-wide desktops.
